Accessibility Statement

Our Commitment

Gamer's Unite! is committed to ensuring our website is accessible to all visitors, including people with disabilities. We believe everyone should be able to browse our overview, learn about what we offer, and find the information they need.

Accessibility Features

We have implemented the following features to improve accessibility:

  • Descriptive alt text on all images, including
  • Color contrast meeting WCAG AA standards (4.5:1 ratio for normal text, 3:1 for large text)
  • Keyboard navigation - all interactive elements can be accessed using keyboard only
  • Semantic HTML structure with proper headings, landmarks, and form labels
  • Responsive design that works on mobile devices and supports zoom up to 200%
  • Visible focus indicators for keyboard users
  • Skip navigation link to jump to main content

Testing

This website has been tested using:

  • WAVE Web Accessibility Evaluation Tool
  • Chrome Lighthouse accessibility audit
  • Manual keyboard navigation testing
  • Chrome DevTools responsive design mode

Each of the pages receieved different Lighthouse results.

Most received a score of 90 or higher and around two of them received a score of 100.

This website was tested on Firefox and it performed well.

Known Limitations

We are aware of the following accessibility limitations and are working to address them:

  • Most of the images have external image as their alt text
  • Some of the navigation is hidden from screen readers even when I tried to fix it multiple times throughout the development process.

Feedback

If you encounter any accessibility barriers while using our website, please let us know. We take accessibility seriously and will work to address issues promptly.

Contact us at: accessibility@gamersunite.example.com
